Instructions

This is a site that allows you to compose a repeating 4 measure G major song in 4/4 time.
Click anywhere on the 8x16 grid to have that note played when the song is started.
You can see the corresponding note and octave below the main grid.
The 1x16 grid represents the 'timestamp' of the song. You manipulate it by clicking anywhere on it.


Will start the song at the current timestamp.

Will stop the song at the current timestamp.

Will stop the song, clear all selected notes, and reset the timestamp.

Determines the number of columns of notes played per second.

Determines the volume of the song.

Determines the type of wave which the note generator will use to generate a note.
